# 商品売上入力・集計プログラム

C# + .NET Framework を使用した、商品名と金額を入力・集計・表示するコンソールアプリケーションです。

## 機能

✅ **複数の商品入力** - 商品名と金額を何件でも入力可能  
✅ **入力値検証** - 金額は正の数値のみを受け付け  
✅ **自動集計** - 以下の値を自動計算
  - 合計金額
  - 平均金額
  - 最高金額
  - 最低金額
  - 入力件数

## システム要件

- Windows
- .NET Framework 4.7.2 以上
- Visual Studio（推奨）または .NET Framework SDK

## ビルド方法

### Visual Studio を使用する場合

1. Visual Studio でプロジェクトを開く
2. ビルド → ソリューションのビルド
3. 実行ファイル `ProductSalesAggregator.exe` が `bin\Debug\` に生成されます

### コマンドラインを使用する場合

```bash
csc Program.cs /target:exe /out:ProductSalesAggregator.exe
```

## 実行方法

```bash
ProductSalesAggregator.exe
```

## プログラムの流れ

1. プログラム起動
2. ユーザーが商品名を入力
3. ユーザーが金額を入力（正の数値のみ）
4. 入力データをリストに追加
5. 続入力するかを確認
6. 入力終了後、合計・平均・最高額・最低額を集計・表示

## エラー処理

- **商品名が空** → エラーメッセージを表示し、再入力を促す
- **金額が数値でない** → エラーメッセージを表示し、再入力を促す
- **金額が負の数** → エラーメッセージを表示し、再入力を促す
